# oystore-banner-review-tool
올영매장 phase3 배너 셀프 검수 도구

올영매장의 배너 이미지를 검수하기 위한 가이드 웹 도구입니다.

## 🔗 바로 사용하기
👉 **[배너 검수 도구 열기](https://purple-kim.github.io/oystore-banner-review-tool/)**


## 📌 주요 기능
### 1. 스토어 썸네일 배너 검수
- **이미지 규격**: 690×516px
- **파일 용량**: 권장 150KB / 최대 300KB
- **자동 검수**: 이미지 사이즈, 파일 용량, 배경색 검증
- 가로형/세로형 비교 뷰 제공
- 안전 영역 자동 표시

### 2. 증정품 이미지 검수
- **관심매장 이미지 규격**: 800×800px (이미지 80×80px)
- **매장행사/바텀시트 이미지 규격**: 800×800px (이미지 96×96px)
- **파일 용량**: 권장 150KB / 최대 300KB
- 안전 영역 가이드 토글
- 해상도별 미리보기

## 🎯 사용 방법
1. 좌측에서 검수할 배너 유형 탭 선택
2. 이미지 파일 업로드 (JPG, PNG) > 드래그 앤 드롭 또는 클릭 (JPG, PNG)
3. 자동 검수 결과 확인
4. (옵션) 증정품의 경우 타이틀 입력 시뮬 통해 텍스트 노출 범위 검수
4. 체크리스트 완료 후 업로드!

## ⚠️ 주의사항
### 스토어 썸네일
- ❌ 배경을 흰색으로 적용하지 마세요
- ✅ 중요한 요소는 중앙 안전 영역(196px)에 배치

### 증정품
- ✅ 배경은 반드시 흰색(#FFFFFF)으로 적용하세요
- ✅ 상품 이미지는 안전 영역 내 배치
